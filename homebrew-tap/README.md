# Homebrew Tap for iOS Health Sync

This is the official Homebrew tap for [iOS Health Sync](https://github.com/mneves75/ai-health-sync-ios).

## Installation

```bash
brew tap mneves75/tap
brew install healthsync
```

## Upgrade

```bash
brew upgrade healthsync
```

## Uninstall

```bash
brew uninstall healthsync
brew untap mneves75/tap
```

## Available Formulae

| Formula | Description |
|---------|-------------|
| `healthsync` | Secure sync of Apple HealthKit data between iPhone and Mac |

## Requirements

- macOS 15 Sequoia or later
- Apple Silicon (arm64) or Intel (x86_64)
